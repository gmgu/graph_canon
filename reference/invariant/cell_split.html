5.6.1. cell_split.hpp¶

Full path: graph_canon/invariant/cell_split.hpp

class invariant_cell_split¶

An implementation of the node invariant consisting of the sequence of cell splits performed in each tree node. It uses the refine_new_cell event to collect data.

Requires a invariant_coordinator as visitor, ordered before this visitor.
