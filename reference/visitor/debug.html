5.15.2. debug.hpp¶

Full path: graph_canon/visitor/debug.hpp

class debug_visitor¶

A visitor for getting debug information.

debug_visitor(bool tree, bool canon, bool aut, bool refine, bool compressed, std::ostream *log = nullptr)¶
Parameters

  • tree – print tree-related debug messages.

  • canon – print debug messages related to the current best leaf.

  • aut – print automorphism-related debug messages.

  • refine – print refinement-related debug messages.

  • compressed – when printing ordered partitions, print only the compressed view.

  • log – print a JSON version of certain log messages to this stream.
