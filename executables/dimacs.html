4.1. dimacs¶

A program for checking the syntax of a graph in DIMACS format (see read_dimacs_graph()), and optionally print a permuted version of it.

-h, --help¶

Print help message.

-f <file>, --file <file>¶

Filename with graph to read. Use - to read form stdin.

-s <seed>, --seed <seed>¶

Seed for random number generator. A default seed from std::random_device will be used if not specified.

--permute¶

Make a random permutation of the graph and print it to stdout.
